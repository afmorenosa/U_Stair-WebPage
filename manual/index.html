Manual de instalación y uso de U Stair

U Stair es una aplicacion que permite a la comunidad universitaria poder contar con un entorno agradable, que organice sus planes a lo largo de sus programas académicos, organizando las materias que ha visto y querrá ver en los diferentes semestres que curse. ¡Comencemos a explorar todas las herramientas con las que cuenta!

Instalación

Para poder instalar la aplicación, haga uso del instalador, segun su sistema operativo.

Windows

Para windows descargue el instalador .exe y ejecutelo, siga las instrucciones del instalador.

Linux

Distros basadas en Debian

Para estas distribuciones puede descargar el archivo .deb. Una vez descargado, desde la terminal ubicarse en el directorio donde se encuentra el instalador y correr alguno de los dos siguientes comandos:

$ sudo apt install ./u_Stair.deb

$ dpkg -i u_Stair.deb

Distros basadas en RPM

El archivo .rpm permite instalar la aplicacion en estas distribuciones, por favor descárguelo y ejecute el siguiente comando desde la terminal, en el directorio donde se descargó el instalador

$ sudo rpm –i u_Stair.rpm

La Aplicación

Una vez la aplicación sea instalada, puede ser ejecutada. Cuando abra la aplicación, aparecerá la ventana de vienvenida, como la que se muestra a continuación.

En esta ventana se podrá crear o abrir archivos propios de la aplicación, los cuales tienen extensión .umsh. También se podrán cambiar la configuración de la aplicacion. Esto se explicará a mayor detalle en las secciones siguientes.
